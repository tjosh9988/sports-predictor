# Bet Hero: AI-Powered Sports Betting Intelligence

Bet Hero is a professional-grade sports prediction and accumulator building platform. It leverages an ensemble of machine learning models to identify high-value bets across multiple sports and markets.

## Core Features

- **ML Ensemble**: Stacking ensemble combining Random Forest, XGBoost, LightGBM, Neural Networks, and LSTM.
- **Value Betting**: Automated EV calculation by stripping bookmaker overrounds.
- **Smart Accumulators**: Three-pass algorithm to build 3-leg, 5-leg, and 10-leg tickets with de-correlation logic.
- **AI Explanations**: Human-readable reasoning for every selection generated by Anthropic Claude (Sonnet 3.5).
- **Automated Pipelines**: Batch training, leakage validation, and backtesting (2 years history).
- **Enterprise API**: FastAPI backend with JWT Auth, WebSockets, and Rate Limiting.

## Architecture

- **Backend**: Python 3.11 (FastAPI)
- **Database**: Supabase (PostgreSQL + Auth)
- **Caching/Queue**: Upstash/Redis
- **ML Tracking**: MLflow
- **Hosting**: Render (Web API + Background Workers)

## Local Setup

### 1. Requirements
- Python 3.11+
- Docker (optional)
- Redis server (local or Upstash)

### 2. Installation
```bash
git clone https://github.com/your-repo/bet-hero-backend.git
cd bet-hero-backend
pip install -r requirements.txt
```

### 3. Configuration
Create a `.env` file in the root:
```env
SUPABASE_URL=your_url
SUPABASE_SERVICE_ROLE_KEY=your_key
REDIS_URL=your_redis_url
ANTHROPIC_API_KEY=your_claude_key
MLFLOW_TRACKING_URI=http://localhost:5000
```

### 4. Run Application
```bash
uvicorn app.main:app --reload
```

## Workflows

### Data Ingestion
```bash
# Import historical data for all sports
python -m app.ingestion.run_importers --all

# Fetch latest fixtures
python -m app.ingestion.fixture_fetcher
```

### Model Training
```bash
# Run the master training pipeline (all sports/markets)
python -m app.ml.training_pipeline --run-all --force
```

### Backtesting
```bash
# Backtest a specific model
python -m app.ml.backtester --sport football --market match_result --years 2
```
